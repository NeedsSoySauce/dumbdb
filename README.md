# Development

These instructions are for people who want to work on dumbdb itself.

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

* [npm](https://www.npmjs.com/)

### Installation

To install dependencies run:

```
npm install
```

### Execution

To run the project:

```
npm run dev
```
